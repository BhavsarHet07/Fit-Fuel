# 💪 Fit Fuel Hub

Fit Fuel Hub is a simple and user-friendly fitness web application designed to help users understand their health condition using basic tools like a BMI calculator.

---

## 🚀 Features

- ⚖️ BMI Calculator – Calculate Body Mass Index بسهولة
- 📊 Instant Results – Get health category (Underweight, Normal, Overweight)
- 🎯 Simple UI – Clean and beginner-friendly interface
- 📱 Responsive Design – Works on mobile, tablet, and desktop
- 🔔 Health Awareness – Helps users understand their fitness level

---

## 🧠 Project Idea

The main idea of Fit Fuel Hub is to provide a **simple and accessible platform** for users who want quick health insights without dealing with complex fitness applications.

---

## ❗ Problem Solved

Many people:
- Don’t understand their health condition
- Find fitness apps too complex
- Need quick and simple health tools

👉 Fit Fuel Hub solves this by offering an easy-to-use BMI-based solution.
